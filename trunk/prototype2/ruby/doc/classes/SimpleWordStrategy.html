Class SimpleWordStrategy
In: etl/stdload.rb
Parent: Object

This class defines the default word strategy adopted when inserting a word into the database

A strategy collects the logic to properly analyze a WordDTO and translate it into the proper database operations. Strategy instances are associated to MysqlETL blocks via their configuration properties.

insert(source_id,page_id,scantime,word)

Inserts a WordDTO into the database. This strategy uses the position attribute of the WordDTO to define the database column to update. Supported positions are : global ,title . Whenever an invalid position is encountered, the word is inserted anyway (with empty counters) and a warning message is issued to the user.
