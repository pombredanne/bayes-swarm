Public Class methods

json_create(o)

Deserializes a JSON object into a proper DTO instance

new(execpointer = 0, dtos = [])

creates a new instance of the class. execpointer is the pointer the first DTO that will be executed when this multiplex will be demuxed in a ChainETL . dtos contains the list of DTOs which compose this multiplex.

Public Instance methods

cur()

returns the DTO currently pointed by the execution pointer

increment_exec(dto)

Increments the execution pointer, passing the dto returned from the last execution

mplex?()

Declares that this DTO supports multiplexing. Anyone which needs to discriminate between DTOs that support multiplexing and the ones that don‘t should test the presence of this method with respond_to? and verify that it returns true .

reset_exec()

resets the execution pointer

to_json(*a)

Serializes this DTO into a JSON object. Refer to the JSON library documentation for further info
