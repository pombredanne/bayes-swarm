close_connection()

Closes the database connection. This method can be safely invoked multiple times.

open_connection(params = {})

Opens a connection to the database. It does not return it (if needed, connections can be retrieved directly from ActiveRecord entries). This method reads parameters both from arguments list and from the @props Hash. Arguments list takes precedence. The relevant keys to be used as params are adapter, host , username , password , database .

with_connection(params={}) {|conn| ...}

Executes the block passed as parameter with an open connection to the database and closes it at the end of the blocks
