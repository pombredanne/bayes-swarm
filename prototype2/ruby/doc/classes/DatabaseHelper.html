Public Instance methods

close_connection()

Closes the database connection. This method can be safely invoked multiple times.

open_connection(params = {})

Opens a connection to the database. It both returns it and stores it the class variable @conn This method reads parameters both from arguments list and from the @props Hash. Arguments list takes precedence. The relevant keys to be used as params are host , user , pass , db .

with_connection(params={}) {|conn| ...}

Executes the block passed as parameter with an open connection to the database. The connection is passed as block parameter. The connection opening happens in the same way as the open_connection method. Connection is always closed at the end of this call.
